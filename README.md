# NC MEMS

Android app for managing MEMS (Micro Electro-Mechanical Systems) equipment.

## Features
- WiFi/BT connection bridge for BLE device access
- QR code MAC address display (scannable from login screen)
- MAC address configuration dialog with real-time validation
- Dark mode support
- Equipment management (JIG, PART, ESD, PRODUCTION EQUIPMENT)

## Build
```bash
./gradlew assembleDebug
```
